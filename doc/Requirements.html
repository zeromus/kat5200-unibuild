Requirements

  • To get full speed - 500 MHz PC (fullscreen 320x240 mode)
  • SDL >= 1.2.0 (Included with Windows Version)
  • SDL_Image >= 1.2.0 (Compiled into Windows Version)
  • Guichan >= 0.8.0 and <= 0.9.0 (Compiled into Windows Version)
  • Zlib >= 1.2.0 (Compiled into Windows Version)
  • Sqlite >= 3.5.0 (Compiled into Windows Version)
